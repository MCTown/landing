2021.8.25

  • 增加了玩家墓碑的功能
    玩家死亡后将生成一个墓碑
    5分钟内只能由该玩家自己打开
    5~15分钟内所有人都可打开
    15分钟后墓碑将自毁并掉落物品（洒落）
    掉岩浆后也会生成墓碑
    墓碑自毁后5分钟后物品永久消失（区块被加载）
    掉落虚空不生成墓碑！
    警告：在墓碑没有摧毁之前即使区块未加载也会计时
  • 修复了击杀bot生成墓碑的BUG
